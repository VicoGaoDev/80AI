# Web 端分步操作指南

本文档基于 `docs/web_user_guide.md` 整理，适合第一次使用 Web 端的普通用户按步骤完成常见操作。

如果你只想快速了解功能概览，也可以查看：

- `docs/web_user_guide.md`：完整功能使用说明
- `docs/web_user_guide_one_page.md`：更短的 1 页版用户手册
- `docs/web_user_guide_screenshot.md`：适合补截图后的培训或客服说明版

## 1. 推荐上手顺序

第一次使用时，建议按照下面顺序操作：

1. 登录或注册账号
2. 浏览首页，了解主要入口
3. 进入模板中心，先挑选接近需求的模板
4. 进入 AI 生图页面开始创作
5. 在结果区预览、下载或继续编辑
6. 到历史记录查看过往作品和完整参数
7. 需要时提交反馈、修改个人资料或查看积分记录

## 2. 第一步：登录或注册

如果你还没有账号，请先注册；如果已有账号，可以直接登录。

操作步骤：

1. 打开 Web 端首页
2. 点击登录或注册入口
3. 输入账号信息并提交
4. 登录成功后进入系统

登录后，你可以使用更多和个人相关的功能，例如：

- 历史记录
- 我的反馈
- 积分记录
- 个人主页

## 3. 第二步：先看首页入口

首页适合用来快速了解平台主要能力，并进入常用页面。

你可以在首页完成：

- 了解平台核心功能
- 找到创作入口
- 找到历史记录入口

如果你还不确定先做什么，建议先从首页进入，再按页面提示继续操作。

## 4. 第三步：优先使用模板中心

如果你不知道怎么写提示词，或者想快速出图，建议先去模板中心。

操作步骤：

1. 进入模板中心
2. 浏览现成模板和展示效果
3. 选择一个接近需求的模板
4. 一键套用模板参数
5. 跳转到创作页面继续修改并生成

模板中心尤其适合下面这些场景：

- 不熟悉提示词写法
- 想先参考别人已经验证过的效果
- 想尽快做出第一张图

## 5. 第四步：进入 AI 生图页面创作

AI 生图页面是最核心的创作区域。你可以根据需求选择不同模式。

### 5.1 文生图

适合从文字描述直接生成图片。

操作步骤：

1. 进入 AI 生图页面
2. 输入提示词
3. 选择模型、比例、尺寸等参数
4. 发起生成任务
5. 在结果区查看生成结果

### 5.2 图编辑

适合基于已有图片继续创作。

操作步骤：

1. 切换到图编辑模式
2. 上传参考图
3. 输入你想调整的方向或风格描述
4. 选择相关参数
5. 提交任务并查看结果

### 5.3 局部重绘

适合只修改图片中的某一部分内容。

操作步骤：

1. 切换到局部重绘模式
2. 上传原图
3. 在需要修改的区域进行涂抹，或准备对应蒙版
4. 输入想替换或新增的内容
5. 提交任务并查看结果

### 5.4 提示词反推

适合参考现有图片的提示词写法。

操作步骤：

1. 切换到提示词反推模式
2. 上传一张参考图片
3. 提交任务
4. 查看系统返回的提示词结果

## 6. 第五步：在结果区处理图片

生成任务完成后，结果通常会展示在结果区。

你可以直接进行以下操作：

- 预览图片
- 下载原图
- 基于结果图继续编辑
- 重新生成
- 提交反馈

如果你得到满意结果，建议及时下载保存。

## 7. 第六步：到历史记录查看过往作品

历史记录适合回看以前生成过的内容，也适合复用参数和继续修改旧作品。

操作步骤：

1. 进入历史记录页面
2. 按提示词等条件筛选记录
3. 查看任务详情和完整参数
4. 重新编辑旧作品，或下载图片
5. 对结果继续提交反馈

如果你经常需要复用以前成功的参数，历史记录会非常有用。

## 8. 第七步：按需使用其他个人功能

除了创作相关页面，Web 端还有几个常用的个人功能页面。

### 8.1 我的反馈

适合提交问题、跟踪处理情况。

你可以：

- 提交使用中的问题或意见
- 查看自己提交过的反馈
- 查看反馈详情和处理进度

### 8.2 个人主页

适合维护个人资料。

你可以：

- 修改用户名
- 上传或更换头像
- 修改登录密码

### 8.3 积分记录

适合查看积分变动原因。

你可以了解：

- 积分增加或减少的记录
- 每次变动的时间
- 对应的用途或来源

## 9. 常用小提示

- 不知道怎么写提示词时，优先使用模板或提示词反推
- 想修改局部内容时，优先使用局部重绘
- 想基于已有图片继续创作时，优先使用图编辑
- 重要图片建议在有效期内及时下载
- 想找回以前成功作品时，优先去历史记录查看

## 10. 常见问题

### 找不到以前生成的图片怎么办？

先到 `历史记录` 中查看。如果图片比较重要，建议在生成后及时下载保存。

### 不知道从哪里开始怎么办？

建议先进入 `模板中心`，选一个接近需求的模板，再进入创作页面继续修改。

### 生成结果不满意怎么办？

你可以尝试：

- 修改提示词后重新生成
- 使用图编辑继续调整
- 使用局部重绘修改局部内容
- 提交反馈说明问题
